kelpie

a little web server in php

What?

Kelpie is a web server for PHP that takes inspiration from the best parts of the Ruby web server ecosystem:

  • Rack, a minimal interface between webservers and Ruby frameworks
  • The Mongrel parser, the root of Mongrel speed and security.
  • Thin, a fast and simple Ruby web server

Installation

You will need to install the httpparser extension. It's a PHP extension for the Mongrel http parser.
